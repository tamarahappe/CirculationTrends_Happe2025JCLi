# Circulation Trends in boreal summer - Happé T. et al., 2025

Main code and figures corresponding to the research article. 

Please note that not all data needed to run the notebooks is available in this repository. Some data is too large, e.g. the ERA5 reanalysis data. Please refer to the data section in the methods of the published paper for more details. 

To cite:
Happé, T., C. van Straaten, R. Hamed, F. D’Andrea, and D. Coumou, 2025: Observed circulation trends in boreal summer linked to two spatially distinct teleconnection patterns. J. Climate, https://doi.org/10.1175/JCLI-D-24-0750.1, in press.
